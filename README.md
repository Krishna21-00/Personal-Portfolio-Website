# Personal Portfolio Website

A modern single-page portfolio website built for Krishna Kumar E.

## Project Name
Personal Portfolio Website

## Features
- Responsive one-page layout
- Smooth scrolling navigation
- Active section highlighting in the navbar
- Mobile-friendly hamburger menu
- Contact form validation
- Download resume button
- Project showcase section

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Open `index.html` in a web browser.
2. Use the navigation bar to jump between sections.
3. Click `Download Resume` to open the resume file.
4. Fill out the contact form to test validation.
